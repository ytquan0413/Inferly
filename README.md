# Inferly Prototype

This folder contains a mobile-first frontend prototype for Inferly, a language learning app focused on contextual reasoning instead of memorization.

## Files

- `index.html` contains the app shell
- `styles.css` contains the visual system and responsive layout
- `script.js` contains the screen flow and interaction logic

## Run

For the most reliable audio playback, run Inferly through a local server instead of opening `index.html` directly.

### Recommended

Double-click `start_inferly.command` in Finder.

This will:

- start a local server at `http://127.0.0.1:8000`
- open Inferly in your browser automatically
- keep the terminal window open while the app is running
